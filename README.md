- Real-Time Analytics

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/asbirvkp/realm-world.git
   ```

2. Install dependencies:
   ```bash
   cd realm-world
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open your browser and visit `http://localhost:3000`

## Development

- The main application logic is in `app.js`
- Frontend JavaScript is in `public/assets/js/script.js`
- Styles are in `public/assets/css/style.css`
- HTML structure is in `public/index.html`
- PNL data is stored in `public/pnlData.json`

To make changes:

1. Edit the relevant files
2. Refresh your browser to see the changes
3. For server-side changes, restart the server with `npm start`
